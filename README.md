* **Full-Stack Web & Management Systems.** I build clean, end-to-end web applications with a focus on business logic and operational efficiency. Specialized in structural refactoring and migrating legacy systems (Native PHP) into modern, maintainable MVC architectures using Laravel.
* **System Architecture & Data Modeling.** I design structural blueprints for software execution. From complex database indexing (MySQL) to multi-level Data Flow Diagrams (DFD), ERD, and UML schemas, ensuring the system design bridges perfectly with the backend code.
